# Vue音乐webapp
>该项目是一个了音乐web app ,使用 Muse-UI 作为项目骨架,才用better-scroll插件来做页面列表滑动,达到接近原生APP一样的效果,对 better-scroll 进行了使用上的封装。


### 1.扫码快速体验（使用手机浏览器打开）
***
![扫码快速体验](http://www.zhengsi.info/1543569162.png)


### 2.安装与运行
***
```
npm install
npm run dev
```

### 3.项目打包
***
```
npm run build
```
